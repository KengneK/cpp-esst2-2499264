
Dieser Repository enthält mehrere Branchs.
- main
- challenge
- solution

Um zwischen Branchs zu navigieren, sollte in Git Batch folgender Befehl eingegeben werden: git checkout branch_name

Nach Bearbeitung wie gewöhnlich speicher: git add.
					  git commit -m 'message'
					  git push

Man sollte aber achten, das man auf dem gewünschten Branch ist.
